◆ maxCarLimit

Integer edu.cu.ooad.Recorder.maxCarLimit = 3
private

The maximum number of Cars that a customer can rent, this is different from per transaction limit.

Per transaction limit specifies what's the maximum (or minimum) number of cars that a given customer can rent in one rental request (transaction). But customer can make multiple rental request, the limit below specifies the maximum number of Cars that certain customer can rent considering all rental requests.
